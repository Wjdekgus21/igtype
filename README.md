# igtype

한국어/영어 타이핑 속도 측정 게임입니다.

## 실행 방법

별도 설치 없이 `igtype html.html` 파일을 브라우저에서 바로 열면 됩니다.

```
igtype html.html 파일을 더블클릭
또는
Chrome / Edge / Safari 등 브라우저에 파일을 드래그 앤 드롭
```

> 인터넷 연결이 필요합니다 (Google Fonts 로드용)

## 파일 구조

```
igtype/
└── igtype html.html   # 전체 게임 (HTML + CSS + JS 단일 파일)
└── README.md
```

## 기능

- **언어 선택** — 한국어 / English
- **난이도 4단계**
  - 쉬움 — 짧은 문장 10개, 1분 30초
  - 보통 — 중간 길이 문장 10개, 1분 30초
  - 어려움 — 긴 문장 10개, 1분 30초
  - 💀 최상 — 초장문 1개 (한국어: ㅅ발음 훈련 / 영어: 전쟁과 평화 발췌), 1분 30초
- **실시간 피드백** — 맞은 글자 초록, 틀린 글자 빨강
- **실시간 통계** — WPM / 정확도 / 진행 현황
- **결과 화면** — 타수(한국어) / WPM(영어), 정확도, 소요 시간, 문장별 속도 그래프
- **최고 기록 저장** — 브라우저 localStorage에 자동 저장

## 개발 환경

바닐라 HTML/CSS/JavaScript — 프레임워크 및 빌드 도구 없음
